# AxiosSample

### Installing


Setting up 

```
1) You will need Node, the React Native command line interface, Python2, a JDK, and Android Studio.
2) After installing all the above install react-native cli using ###npm install -g react-native-cli
3) Create your apllication using ###react-native init AxiosSample
4) Add axios as a dependency in package.json file.
5) cd AxiosSample
5) Install all the modules using npm install
6) Copy all the code from App.js to your newly created App.js file.
7)Run react-native run-android(Make sure the android emulator is running)

For more reference-https://facebook.github.io/react-native/docs/getting-started.html
